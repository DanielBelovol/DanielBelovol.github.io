[View on GitHub](https://github.com/DanielBelovol/ThumbnailTester)

**ThumbnailTester** is a commercial project for testing YouTube thumbnails and analyzing video statistics.  
I actively participated in the backend development of this project, which included:

- Designing backend services and database schemas
- Collecting data from YouTube API and processing it
- Implementing dynamic A/B testing with WebSocket-based real-time updates
- Managing user authentication and configuration for test campaigns
- Ensuring scalability and reliability of the system

**Key Features:**

- Run A/B tests for thumbnails and titles
- Track detailed statistics: views, average watch duration, CTR, watch time
- Dynamic test configuration via WebSocket API
- Support for Google authorization and analytics integration
